# HomePage

浏览器自定义首页项目。

## 项目概述

一个纯静态 HTML 单页应用，作为浏览器的新标签页首页使用。

**功能模块：**
- 搜索框
- 书签（快捷链接）
- 待办事项

## 技术栈

- 纯 HTML / CSS / JavaScript，不使用任何框架或构建工具
- 单文件或极少数文件，直接 `file://` 协议打开

## 数据持久化

所有配置和用户数据统一存储在 `localStorage` 中，以 JSON 格式管理：

```json
{
  "bookmarks": [...],
  "todos": [...],
  "settings": {...}
}
```

**设计原则：**
- 所有数据通过一个统一的配置对象管理，方便整体导出/导入迁移
- 提供导出为 JSON 文件和从 JSON 文件导入的功能

## 部署

本地使用，直接用浏览器打开 `index.html` 即可，无需服务器。
