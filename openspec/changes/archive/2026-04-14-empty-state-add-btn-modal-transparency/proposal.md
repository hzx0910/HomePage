## Why

书签为空时显示纯文字提示「暂无书签」，缺乏直接操作入口，用户需先找到添加按钮才能开始使用。同时，添加书签和添加备忘的弹窗背景为不透明深色，与搜索推荐词区域的半透明风格不一致，视觉层次感不足。

## What Changes

- 书签空态：移除「暂无书签」纯文字提示，替换为可点击的「＋ 添加书签」按钮，点击后打开添加书签弹窗
- 弹窗透明度：将添加书签弹窗和添加备忘弹窗的背景改为与搜索推荐词区域相同的半透明样式（`--modal-bg` CSS 变量）

## Capabilities

### New Capabilities

（无新能力，均为现有能力的行为调整）

### Modified Capabilities

- `bookmarks`: 书签空态行为变更——从静态文字提示改为可操作的添加按钮
- `todos`: 弹窗背景透明度调整——与搜索推荐区域风格对齐

## Impact

- `index.html`：修改 `Bookmarks.render()` 空态逻辑；调整 `.modal` CSS 背景属性
- 不影响 localStorage 数据结构或任何业务逻辑
