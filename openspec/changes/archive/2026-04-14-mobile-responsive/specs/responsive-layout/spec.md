## ADDED Requirements

### Requirement: 窄屏搜索区单列布局
当视口宽度 ≤ 600px 时，搜索表单 SHALL 切换为单列布局：搜索输入框占满全宽，引擎按钮组在其下方单独一行展示。

#### Scenario: 窄屏搜索框全宽
- **WHEN** 视口宽度 ≤ 600px
- **THEN** 搜索输入框宽度占满内容区全宽，引擎按钮组显示在搜索框正下方

#### Scenario: 宽屏恢复双列
- **WHEN** 视口宽度 > 600px
- **THEN** 搜索表单恢复为 `3fr 1fr` 双列，搜索框与引擎按钮组并排显示

### Requirement: 窄屏主内容区单列布局
当视口宽度 ≤ 600px 时，主内容区 SHALL 切换为单列布局，书签卡片在上，待办卡片在下（若待办未隐藏）。

#### Scenario: 窄屏待办堆叠在书签下方
- **WHEN** 视口宽度 ≤ 600px 且待办区域未隐藏
- **THEN** 书签卡片显示在上方，待办卡片紧接其下方，均占满全宽

#### Scenario: 窄屏隐藏待办时书签全宽
- **WHEN** 视口宽度 ≤ 600px 且 `settings.showTodos` 为 false
- **THEN** 书签卡片占满全宽，待办卡片不显示（与宽屏隐藏逻辑一致）

#### Scenario: 宽屏恢复双列主内容
- **WHEN** 视口宽度 > 600px
- **THEN** 主内容区恢复 `3fr 1fr` 双列，书签与待办并排显示

### Requirement: 窄屏边距收窄
当视口宽度 ≤ 600px 时，页面 body 水平内边距 SHALL 收窄，保证内容不贴近屏幕边缘。

#### Scenario: 窄屏内边距适配
- **WHEN** 视口宽度 ≤ 600px
- **THEN** body 水平内边距不大于 12px，垂直方向顶部内边距适当减小
